# AI Agent Rules

A collection of rules and guidelines for AI agents to follow when generating or reviewing code.

## Repository Structure

The rules are currently organized into categories within the `code-rules` directory, with more categories planned for the future beyond just coding rules.

### [General Rules](file:///f:/Github-Repository/AI-Agent-Rules/code-rules/general)

This directory contains fundamental rules that apply across most coding tasks.


## Usage

AI agents should read these rules at the start of a task and adhere to them throughout the development process. These rules are designed to ensure consistency, quality, and maintainability across all AI-generated or AI-reviewed code.

---

*This repository is a live document and rules may be updated as best practices evolve.*
